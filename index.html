Projects

Some of the projects I've worked on since I started my Coding bootcamp. You can access my repositories following the Github link in each section.

  • Star Wars Dashboard

    This project was developed following the tutorial from the Youtube Channel Papo de Dev. It was a really interesting shadowing project to understand more about Javascript and how to structure elements in the page.

    Although I did not created the layout and code myself I still tried to understand the bits of code to acquire as much knowledge as possible. To see the source code access my repository.

  • SunnySide Landing Page

    The Sunnyside project is a fictional agency Landing page developed using CSS and HTML only. It was really challenging at first and even thought I only work in this project for a week it gave me some insights about how to structure a web page and organize a project.

    Taking this challenge helped me to understand a bit more about CSS Flexbox and Grid Layout and get a sense of how the properties work with parent and child containers. I also tried to develop a mobile version of the page. You can check more about it in my Github repo!

  • Mixed Messages

    Mixed Messages is a random advice/fortune teller generator using only JavaScript. The messages are randomly generator and some parts of the messages change each time. The goal of the project was to create a program that returns a new message each time the user reloads the program.

    The program is simple and it logs the messages to the console however it was very useful to practice key concepts of objects and arrays.
